SC-401 Lab 1 – Assign Compliance and Security Roles

Assign Microsoft 365 compliance and security roles to a designated user to enable access to Purview and Defender portals.

Home Projects Scripts Home-Lab Config Labs Contact

Tools Used

  • Microsoft Learn Sandbox
  • Microsoft 365 Admin Center
  • Microsoft Purview Portal
  • Microsoft Defender Portal

Steps Taken

  1. Signed in as MOD Administrator in the Microsoft 365 Admin Center.
  2. Selected user Joni Sherman from Active Users.
  3. Assigned the following roles:
    • Compliance Administrator
    • Security Administrator
    • Application Administrator
  4. Confirmed role assignment and access permissions.
  5. Signed in as Joni Sherman to verify access to Microsoft Purview and Defender portals.

Outcome

Role-based access successfully configured. Joni Sherman gained access to Microsoft Purview and Defender portals, enabling delegated administration for compliance and security workflows.

Certification Alignment

SC-401 Domain 1.1 – Manage Microsoft 365 Role-Based Access
