SC-401 Lab 9 – Configure DSPM for AI

Deploy Data Security Posture Management to monitor and protect sensitive data used by AI applications across Microsoft 365.

Home Projects Scripts Home-Lab Config Labs Contact

Tools Used

  • Microsoft Learn Sandbox
  • Microsoft Purview Portal
  • Microsoft 365 Compliance Center
  • Microsoft Defender Portal

Steps Taken

  1. Signed into Microsoft Purview portal as MOD Administrator.
  2. Navigated to Data Security Posture Management → AI Data Protection.
  3. Enabled DSPM and reviewed AI data access patterns.
  4. Created a policy to monitor sensitive data used by Copilot and other AI tools.
  5. Configured alerts and remediation actions for unauthorized access or sharing.
  6. Simulated AI-driven data access and validated policy enforcement.
  7. Reviewed audit logs and risk insights in Microsoft Defender portal.

Outcome

DSPM for AI successfully configured and validated. Microsoft 365 now monitors sensitive data interactions with AI tools, supporting secure deployment and responsible AI governance.

Certification Alignment
