SC-401 Lab 0 – Environment Setup

Prepare your Microsoft 365 environment for SC-401 labs by enabling auditing, device onboarding, insider risk analytics, Defender XDR, and MFA.

Home Projects Scripts Home-Lab Config Labs Contact

Tools Used

  • Microsoft Learn Sandbox
  • Microsoft Purview Portal
  • Microsoft Defender Portal
  • Microsoft Entra Admin Center

Steps Taken

  1. Launch Microsoft Learn sandbox environment.
  2. Navigate to Microsoft Purview → Enable Audit.
  3. Navigate to Microsoft Defender → Enable Device Onboarding.
  4. Go to Insider Risk Management → Enable Analytics and Data Sharing.
  5. Open Microsoft Defender XDR → Confirm access and initialization.
  6. Navigate to Microsoft Entra Admin Center → Enable Multi-Factor Authentication.

Outcome

Environment successfully prepared for SC-401 labs. All core services activated and ready for policy configuration and compliance testing.

Certification Alignment

SC-401 Domain 1.1 – Plan and implement Microsoft Purview solutions
