SC-401 Lab 3 – Create and Publish Sensitivity Labels

Design and deploy sensitivity labels to classify and protect organizational data across Microsoft 365 services.

Home Projects Scripts Home-Lab Config Labs Contact

Tools Used

  • Microsoft Learn Sandbox
  • Microsoft Purview Portal
  • Microsoft 365 Compliance Center

Steps Taken

  1. Signed into Microsoft Purview portal as MOD Administrator.
  2. Navigated to Information Protection → Labels.
  3. Created a new sensitivity label named Confidential – Internal Only.
  4. Configured encryption settings and content marking options.
  5. Published the label via a new label policy targeting all users.
  6. Verified label availability in Outlook and Word web apps.

Outcome

Sensitivity label successfully created and published. Users can now classify and protect content using built-in Microsoft 365 labeling tools, supporting compliance and data governance objectives.
