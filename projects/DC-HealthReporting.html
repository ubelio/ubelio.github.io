Challenge

To gain centralized visibility into domain controller health, replication status, and performance trends, I implemented Microsoft Entra Connect Health for AD DS. This solution provides dashboards, alerts, and graphical insights directly within the Entra Admin Center.

Tools & Technologies

  • Microsoft Entra Connect Health Agent
  • Entra Admin Center
  • AD DS (Windows Server 2012 R2 and above)
  • Performance Counters & Alerting Blades
  • Domain Controller Grouping by Site/Domain

Implementation

  1. Installed Entra Connect Health Agent on all domain controllers
  2. Verified agent registration and data flow to Entra Admin Center
  3. Enabled alert notifications for replication failures, service outages, and performance degradation
  4. Configured dashboards for Domain Controller Health and Replication Status
  5. Customized performance counter views for CPU, memory, and replication latency
  6. Grouped domain controllers by site for topological clarity
