🔐 Microsoft Secure Score Optimization: 40% → 87%

Overview:
Led a comprehensive security hardening initiative that elevated Microsoft Secure Score from 40% to 87%, aligning with industry best practices and compliance goals. This project involved strategic deployment of identity protection, access governance, and password hygiene across hybrid environments.

Key Actions Taken:

  • ✅ Enforced Multi-Factor Authentication (MFA) for all users
  • 🔐 Created Conditional Access Policies for risky sign-ins and risky users
  • 🔄 Enabled Self-Service Password Reset (SSPR) for secure password remediation
  • 🛡️ Deployed Microsoft Defender for Identity across domain controllers
  • 🧠 Implemented Entra Password Protection for cloud and on-prem environments via AD agents
  • 👥 Activated Privileged Identity Management (PIM) for just-in-time role access
  • 🔍 Scheduled Access Reviews to enforce least privilege and role hygiene

Impact:

  • 📈 Secure Score increased by 47 percentage points
  • 🔄 Reduced exposure to identity-based threats
  • 🧩 Strengthened hybrid identity posture across Microsoft 365 and on-prem AD
  • 🧭 Improved audit readiness and governance transparency

Technologies Used:

  • Microsoft Entra ID (formerly Azure AD)
  • Microsoft Defender for Identity
  • Microsoft 365 Security & Compliance Center
  • PowerShell (for policy auditing and reporting)

Relevant Certification:
SC-300: Microsoft Identity and Access Administrator
