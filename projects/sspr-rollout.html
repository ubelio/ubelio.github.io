SSPR Rollout

Self-Service Password Reset with Hybrid AD and Third-Party Sync

Challenge

Our helpdesk was overwhelmed with password reset tickets. While a ManageEngine solution existed, it only worked well for on-prem users. Remote users had to restart or reset passwords twice. We needed a seamless, secure, and modern solution that worked from anywhere.

Tools & Technologies

  • Microsoft Entra ID (SSPR)
  • Microsoft Entra Connect (Password Writeback)
  • ManageEngine Password Sync Agent
  • Group Policy Objects (GPO)

Implementation

  1. Configured SSPR in Microsoft Entra ID:
    • Authentication methods
    • Registration and notifications
    • Targeted Entra security groups
  2. Enabled password writeback in Microsoft Entra Connect to sync cloud password changes to on-prem AD.
  3. Configured ManageEngine agents on domain controllers to sync AD password changes to third-party apps (IBM, XA, etc.) within 5 minutes.
  4. Communicated changes to users and deployed reset links via GPO to desktops and browser favorites.
