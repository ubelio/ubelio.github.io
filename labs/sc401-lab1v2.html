SC-401 Lab 2 – Create and Customize Sensitive Information Types

Define and tailor sensitive information types to support data classification and protection policies in Microsoft Purview.

Home Projects Scripts Home-Lab Config Labs Contact

Tools Used

  • Microsoft Learn Sandbox
  • Microsoft Purview Portal
  • Microsoft 365 Compliance Center

Steps Taken

  1. Signed into Microsoft Purview portal as MOD Administrator.
  2. Navigated to Data Classification → Sensitive Info Types.
  3. Created a new custom sensitive information type named Ubelio Custom ID.
  4. Defined a keyword match and regular expression pattern.
  5. Configured confidence levels and proximity settings.
  6. Saved and published the new sensitive information type.
  7. Verified availability for use in DLP and auto-labeling policies.

Outcome

Successfully created and customized a sensitive information type tailored to organizational needs. This enables more precise data classification and supports downstream DLP and labeling workflows.
