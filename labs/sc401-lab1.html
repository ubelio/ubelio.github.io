SC-401 Lab 1 – Assign Compliance and Security Roles

Certification Domain: SC-401 Domain 1.1 – Manage Microsoft 365 Role-Based Access

Completion Method: GitHub lab walkthrough and screenshot analysis

This lab was completed via GitHub lab review. No tenant access was used.

Objective

Assign appropriate compliance and security roles to a designated user (Joni Sherman) to enable access to Microsoft Purview and security tools. This lab demonstrates role-based access control and administrative delegation within Microsoft 365.

Walkthrough Summary

  • Logged into Microsoft 365 Admin Center as MOD Administrator
  • Navigated to Active Users and selected Joni Sherman
  • Assigned the following roles:
    • Compliance Administrator
    • Security Administrator
    • Application Administrator
  • Verified role assignment confirmation
  • Signed in as Joni Sherman to explore the Microsoft Purview portal

Impact Metrics

  • ✅ Role-based access configured for compliance and security workflows
  • ✅ Microsoft Purview portal successfully accessed
  • ✅ Demonstrates SC-401 Domain 1.1 alignment with least privilege and delegated administration
