Initial Firewall Deployment & Access Control

Deploy OPNsense firewall, validate WAN/LAN connectivity, confirm anti-lockout safeguards, and restrict administrative access.

Home Projects Scripts Home-Lab Config Labs Contact

Tools Used

  • OPNsense Firewall (latest stable release)
  • Web GUI (HTTPS access)
  • LAN/WAN interfaces
  • 2FA for admin access (optional)

Steps Taken

  1. Installed OPNsense on dedicated hardware/VM.
  2. Verified WAN connectivity (public IP assigned, outbound reachability).
  3. Verified LAN connectivity (clients able to reach firewall GUI).
  4. Confirmed anti-lockout rule exists by default (ensures GUI access from LAN).
  5. Restricted GUI access to LAN only (disabled WAN GUI exposure).
  6. Enabled 2FA for admin login to strengthen access control.
