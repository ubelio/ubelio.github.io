GeoIP Blocking

Block traffic from selected geographic regions to enforce compliance and reduce attack surface.

Home Projects Scripts Home-Lab Config Labs Contact

Tools Used

  • OPNsense Firewall
  • Firewall Aliases (GeoIP)
  • Firewall Rules (WAN → LAN)
  • Log Viewer
  • Diagnostics: External reachability tests

Steps Taken

  1. Created GeoIP alias blocking selected countries.
  2. Applied alias in WAN rule to block inbound traffic.
  3. Tested external connections from blocked regions.
  4. Confirmed traffic denied and logged.
  5. Verified traffic from allowed regions remained functional.
