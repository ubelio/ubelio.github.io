Outbound Rule Hardening

Restrict outbound traffic to essential services (HTTPS/DNS) and block/log all other traffic.

Home Projects Scripts Home-Lab Config Labs Contact

Tools Used

  • OPNsense Firewall
  • Firewall Rules (LAN → WAN)
  • Diagnostics: Ping/Traceroute
  • Log Viewer

Steps Taken

  1. Created explicit allow rules for HTTPS (TCP/443) and DNS (UDP/53).
  2. Added final block/log rule for all other LAN → WAN traffic.
  3. Validated HTTPS browsing and DNS lookups worked.
  4. Confirmed non-allowed outbound traffic was blocked and logged.
