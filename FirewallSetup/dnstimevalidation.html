DNS & Time Sync Validation

Ensure reliable DNS resolution and accurate time synchronization before applying security policies.

Home Projects Scripts Home-Lab Config Labs Contact

Tools Used

  • OPNsense Firewall
  • Unbound DNS Resolver
  • System Time Sync (NTP)
  • Diagnostics: DNS Lookup, Ping, NTP Status

Steps Taken

  1. Verified DNS resolution using Diagnostics → DNS Lookup (queried microsoft.com).
  2. Confirmed upstream DNS servers were reachable via Diagnostics → Ping.
  3. Ensured Unbound DNS Resolver was active and listening on LAN.
  4. Checked System → Settings → General for correct DNS server entries.
  5. Validated NTP sync via System → Settings → Time and Diagnostics → NTP Status.
  6. Confirmed system time was accurate and synchronized with upstream NTP servers.
