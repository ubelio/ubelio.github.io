Communication Compliance

Detect and remediate inappropriate text messages using Microsoft Purview Communication Compliance policies scoped to E5-licensed users.

Home Projects Scripts Home-Lab Config Labs Contact

Tools Used

  • Microsoft Purview Communication Compliance
  • Microsoft 365 E5
  • Microsoft Teams & Exchange Online
  • Built-in Classifiers: Inappropriate Text, Content that Matches: Sexual, Hate, Violence, Self Harm
  • Trainable Classifiers: Targeted Harassment, Threat, Discrimination, Adult images, Racy images

Steps Taken

  1. Created Communication Compliance policy using built-in "Inappropriate Text" classifier.
  2. Scoped policy to E5-licensed users only.
  3. Sent simulated inappropriate messages via Teams.
  4. Monitored Communication Compliance dashboard for alert generation.
  5. Validated detection latency and classifier accuracy.
